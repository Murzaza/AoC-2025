# Advent of Code 2025

## Run

```bash
gleam test # Runs the test files
gleam run -- -d <day> -p <part> # Runs day and part. Both are optional
gleam run -- -d 2 # will run both parts of day 2
gleam run -- -d 2 -p 1 # will run day 2 part 1
gleam run # Will run all parts
```
